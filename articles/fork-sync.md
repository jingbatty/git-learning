### 步骤
- 在喜欢的仓库，点击`fork`
- 把自己github仓库，`clone`到本地
- 查看原始仓库 `git remote -v`
- 添加上游仓库 `git remote add upstream https://github.com/humingx/club-doc.git`
- 同步fork仓库 `git fetch upstream`
- 合并提交 `git merge upstream/master`
